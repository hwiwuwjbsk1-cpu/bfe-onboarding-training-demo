# 出口易新人入职网页小游戏大厅

这是一个用于新人培训方向探索的网页版小游戏项目。当前版本重点不是把 PPT 内容硬塞进页面，而是先把“能直接玩”的小游戏大厅做出来，再轻量融入新人场景。

## 当前版本

- 清爽网页游戏大厅
- 10 个小游戏入口
- 小游戏本身不绑定培训文档，先保证玩法轻松直观
- 每个小游戏通关后弹出图文介绍，不跳转到单独学习页
- 第一关通关弹窗为“公司的起源”
- 节奏大师：看准轨道节拍，点中亮起音符
- 排雷探图：翻开星星格，避开隐藏炸弹
- 数字华容道、记忆翻牌、反应速度、疯狂点击等基础小游戏
- 金币、最高分、游玩次数统计
- 本地进度保存
- 支持 `?game=rhythm`、`?game=minefield` 这类直达关卡参数
- 支持静态打包部署，本地打开不需要外网

## 资料来源

小游戏玩法不直接使用内部培训材料。通关后的弹窗介绍从 PPT 中提取真实公司介绍、图片和新人说明材料。

## 技术栈

- React
- Vite
- TypeScript
- lucide-react
- Phaser 扩展底座已保留，后续可接入更复杂的 2D 场景

## 本地运行

```bash
npm install
npm run dev-nolog
```

默认地址：

```text
http://localhost:8080
```

## 线上仓库

公开仓库：

```text
https://github.com/hwiwuwjbsk1-cpu/bfe-onboarding-training-demo
```

推送到 `main` 后，GitHub Actions 会自动安装依赖、打包 `dist`，并发布到 GitHub Pages。

## 打包

```bash
npm run build-nolog
```

产物目录：

```text
dist
```

`dist` 目录可以直接压缩后上传服务器测试。

## 本地预览说明

- 本机 `http://127.0.0.1:端口/` 不需要外网。
- GitHub Pages 或 GitHub 仓库链接需要外网。
- 普通静态服务器只需要上传 `dist` 目录。
